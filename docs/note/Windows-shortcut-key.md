# Windows快捷键

## 1. 常用快捷键

> **复制**：Ctrl+C
>
> **粘贴**：Ctrl+V
> 
> **剪切**：Ctrl+X
> 
> **撤销**：Ctrl+Z
> 
> **全选**：Ctrl+A
> 
> **保存**：Ctrl+S
> 
> **查找**：Ctrl+F
> 
> **替换**：Ctrl+R
> 
> **打印**：Ctrl+P

## 2. 常规操作

> **从右往左删除内容**：Backspace
> 
> **从左往右删除内容**：Delete
> 
> **使鼠标光标处于当前行行首**：Home
> 
> **使鼠标光标处于当前行行尾**：End
> 
> **删除文件至回收站**：Delete
> 
> **删除文件且不放入回收站（永久删除）**：Shift+Delete
> 
> **打开运行窗口**：Windows+R
> 
> **打开文件资源管理器**：Windows+E
> 
> **打开任务管理器**：Ctrl+Alt+Delete 或 Ctrl+Shift+ESC
> 
> **关闭当前窗口 或 网页**：Alt+F4

> 注：任务管理器里的Windows资源管理器用于显示桌面，若结束该任务，则桌面消失，需在任务管理器中新建任务，输入explorer即可重新显示桌面。
